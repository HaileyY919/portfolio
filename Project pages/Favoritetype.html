Favorite Type Poster

In my typeography class we were assigned a project where we had to make an entire poster only using type. On top of this, it had to be our "favorite typeface" and we could only use that singular typeface. I chose to do Clarendon after some research because I thought it would be fun to create a wanted poster, which Clarendon was commonly used for when it was first created by Robert Besley in 1845.
